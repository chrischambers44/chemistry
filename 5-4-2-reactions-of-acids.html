Core ideas for 5.4.2

Most exam questions on acids boil down to three big patterns: (1) acids + metals → salt + hydrogen, (2) acids + bases/alkalis → salt + water (neutralisation), and (3) choosing the right method to make a salt.

Key definitions

  • Acid: produces H+ ions in water
  • Base: neutralises acids (often metal oxides/hydroxides/carbonates)
  • Alkali: a soluble base (e.g. NaOH)
  • Neutralisation: H+ + OH− → H2O
  • Salt: ionic compound formed when H+ in an acid is replaced by a metal (or NH4+)

5.4.2.1 Acids + metals

General word equation:
acid + metal → salt + hydrogen
Example: magnesium + hydrochloric acid → magnesium chloride + hydrogen
Mg + 2HCl → MgCl2 + H2
Observations:
  • Fizzing/bubbles (hydrogen gas)
  • Metal dissolves (often)
  • Solution warms (exothermic)
Test for hydrogen: lighted splint → squeaky pop.

5.4.2.2 Neutralisation (acids + bases/alkalis)

General word equation:
acid + base → salt + water
Ionic equation (the core idea):
H+ + OH− → H2O
Examples:
  • HCl + NaOH → NaCl + H2O
  • H2SO4 + CuO → CuSO4 + H2O

Acids + carbonates (very common)

acid + carbonate → salt + water + carbon dioxide
Example: 2HCl + CaCO3 → CaCl2 + H2O + CO2
Test for carbon dioxide: bubble through limewater → turns milky/cloudy.

5.4.2.3 Making salts: which method?

The “right” method depends on whether the base is soluble or insoluble.
GoalUse this methodKey steps / keywords
Make a salt from an insoluble base (e.g. CuO) Excess insoluble base + acid Add excess base → filter → crystallise
Make a salt containing Na⁺ / K⁺ / NH₄⁺ (all soluble) Titration (acid + alkali) Indicator → repeat without indicator → crystallise
Method: excess insoluble base (example CuSO₄)
  1. Warm acid (e.g. dilute sulfuric acid).
  2. Add excess copper oxide until no more reacts.
  3. Filter to remove leftover solid.
  4. Evaporate some water, leave to cool → crystals form.
  5. Filter and dry crystals.
Method: titration (example NaCl)
  1. Use a pipette to add a measured alkali (e.g. NaOH) into a flask.
  2. Add a few drops of indicator.
  3. Add acid from a burette until the end-point colour change.
  4. Repeat using the same volumes but without indicator.
  5. Evaporate and crystallise.

Common mark-scheme phrases (copy-friendly)

These are the “safe phrases” that match typical AQA mark schemes.

Patterns

“Acid + metal produces a salt and hydrogen.”
“Neutralisation is when an acid reacts with a base to form a salt and water.”
“The ionic equation for neutralisation is H+ + OH− → H2O.”

Gas tests

“Hydrogen is identified using a lighted splint; it burns with a squeaky pop.”
“Carbon dioxide turns limewater milky/cloudy.”

Making salts

“An insoluble base is added in excess so all the acid is neutralised.”
“The mixture is filtered to remove excess solid, then the filtrate is crystallised to obtain the salt.”
“Titration is used when both reactants are soluble because you can’t filter off excess.”
